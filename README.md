# 📊 Customer Churn Analysis

A data analytics project that explores customer churn patterns in a telecom company using Python. The project focuses on identifying the factors that influence customer churn through data cleaning, exploratory data analysis (EDA), and visualization, followed by actionable business recommendations.

---

## 📌 Project Overview

Customer churn is one of the biggest challenges for subscription-based businesses. Understanding why customers leave helps organizations improve customer retention and reduce revenue loss.

This project analyzes a telecom customer dataset to uncover key churn drivers and provide insights that support better business decisions.

---

## 🎯 Objectives

- Clean and preprocess the dataset
- Perform Exploratory Data Analysis (EDA)
- Visualize customer churn patterns
- Identify the major factors contributing to churn
- Generate business recommendations to improve customer retention

---

## 🛠️ Tools & Technologies

- Python
- Pandas

- Matplotlib

---

## 📂 Dataset Information

- **Records:** 7,043 Customers
- **Features:** 21
- **Domain:** Telecom Customer Churn

The dataset contains customer demographics, account information, services subscribed, billing details, and churn status.

---

## 🔍 Data Preprocessing

The following preprocessing steps were performed:

- Converted `TotalCharges` to numeric format
- Handled missing values
- Checked for duplicate records
- Converted binary values into readable categories
- Verified data consistency before analysis

---

## 📊 Exploratory Data Analysis

The analysis includes visualizations for:

- Overall customer churn
- Gender-wise churn
- Senior citizen churn
- Customer tenure analysis
- Contract type analysis
- Internet service analysis
- Online security & tech support analysis
- Streaming services analysis
- Payment method analysis

---

## 📈 Key Insights

- Approximately **26.5%** of customers have churned.
- Customers with **Month-to-Month contracts** show the highest churn rate.
- Customers in the **early months of their subscription** are more likely to leave.
- **Fiber Optic** users experience higher churn than DSL users.
- Customers without **Online Security** and **Tech Support** churn significantly more.
- Customers using **Electronic Check** as their payment method have the highest churn.

---

## 💡 Business Recommendations

- Improve onboarding for new customers.
- Encourage long-term contracts through attractive offers.
- Promote value-added services such as Online Security and Tech Support.
- Develop targeted retention strategies for high-risk customer segments.
- Encourage customers to switch to automatic payment methods.
